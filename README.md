# SenacOS · .github

Este é o repositório especial de configuração da organização SenacOS. Os arquivos aqui não são exibidos como parte de um projeto — eles funcionam como padrão automático para qualquer outro repositório da organização que não tenha sua própria versão de um arquivo equivalente.

> 💡 **Nota:** Se você está procurando a página inicial da organização (a que aparece em `github.com/SenacOS`), ela vive em `profile/README.md`, não neste arquivo.

## 📂 O que tem aqui

| Arquivo / Diretório | Para que serve |
| :--- | :--- |
| `.github/CONTRIBUTING.md` | Resumo rápido de como contribuir (fluxo de PR, Issues, projetos acadêmicos). |
| `.github/CODE_OF_CONDUCT.md` | Regras de convivência da comunidade. |
| `.github/SUPPORT.md` | Onde buscar ajuda (Discussions, Issues, times por curso). |
| `.github/PULL_REQUEST_TEMPLATE.md` | Template padrão carregado automaticamente em todo Pull Request. |
| `.github/ISSUE_TEMPLATE/` | Templates de Issue (`bug_report`, `feature_request`) + `config.yml`. |
| `profile/README.md` | Landing page pública da organização. |

## 🔗 Documentação completa

O guia detalhado de contribuição — modelo de membresia (contribuidor x membro), Teams por curso, nomenclatura de projetos acadêmicos e o ecossistema completo de repositórios — vive no repositório **[SenacOS/core](https://github.com/SenacOS/core)**.

---
*⚠️ **Atenção:** Qualquer alteração aqui afeta toda a organização. Repositórios sem um arquivo próprio do mesmo tipo herdam automaticamente o que está neste repositório. Revise com atenção antes de dar merge.*
